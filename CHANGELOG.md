# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of JSON Restruct
- Core mapping functionality with wildcard support
- Multiple built-in transforms (toNumber, toDateTime, case conversions, regex, etc.)
- Conditional transforms with when/then syntax
- Row-level filtering capabilities
- Array filtering with [key=value] syntax
- Nested output schemas
- Multi-path fallback support
- Schema validation
- Comprehensive TypeScript definitions
- JSDoc documentation throughout codebase
- Professional README with examples
- Test suite with multiple test cases
- Example files demonstrating various use cases

### Features
- High-performance JSON Schema Mapper
- Support for Excel-style column names with bracket notation
- Transform pipelines for chained transformations
- Regex capture groups with group indexing
- Default values for missing data
- Memory-efficient processing for large datasets (100k+ rows)
